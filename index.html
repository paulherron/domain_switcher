Some handy bookmarklets for toggling domains

When clicked, this bookmarklet will take you from a .com domain to a .l, and vice versa. This is handy for jumping between the same page on the live and local versions of the site.

You can also make the bookmarklet a 'keyword bookmark'. I've assigned ! as the keyword, so to do the toggle I can just type Cmd-l (to get the address bar) and then !<Enter>. The keyword bookmark also takes an optional argument to go to a specified TLD, e.g. ! org.

TLD Switcher bookmarklet

This bookmarklet simply removes a subdomain portion of the current URL. So for example it would take you from dev.showstudio.com to showstudio.com.
